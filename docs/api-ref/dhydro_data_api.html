clip_structures_by_branches(buffer: float = 1, features: Optional[list] = None, min_overlap: float = 0.5) → None[source]#

Class method to drop structures from the data that are farther from a branch than “buffer” Structures that are linestring are required to overlap with a branch for at least “min_overlap”

Parameters:

  • buffer (float) – distance from branch that structures are kept (default = 1m)

  • min_overlap (float) – percentage of branch and LineString structure that should overlap in order to keep it. this drop, for example, culverts that are perpendicular to a branch

Returns:

None
